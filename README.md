# Snake Water Gun Game (Python)

This is a simple **Snake, Water, Gun** game written in **Python**.  
It is a beginner-friendly project made while practicing Python basics.

---

## What This Game Does

- The computer randomly chooses:
  - Snake
  - Water
  - Gun
- You enter your choice using:
  - `s` for Snake
  - `w` for Water
  - `g` for Gun
- The game compares your choice with the computer’s choice
- Then it tells you whether you **win**, **lose**, or **draw**

---

## Rules of the Game

- Snake drinks Water → **Snake wins**
- Water damages Gun → **Water wins**
- Gun kills Snake → **Gun wins**
- Same choice → **Draw**

---

## How to Run the Game

1. Download or clone the repository
2. Open the terminal in the project folder
3. Run the file:
